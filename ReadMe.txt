Create Portfolio website.

Steps-
1.Download all the files and save them in htdocs if you are using Xampp.
2.Host the site on a Local server using Xampp preferably.
3.Create 'photos' folder to store profile pictures.
4.Create Mysqli Database named portfolio and the create table 'form' as given in sql file.
5.On Google console register yourself to get the Client_id and client_secret and set redirect domain to-'http://localhost/googlelogin.php'
6.Inside google_signup folder,in settings.php enter your Client_id and Client_secret.
7.Start xampp.
8.Open port-login.php file.

There you go, now you can log in and create portfolio and can even share them just by sharing your username.
